# Roll Dice

A Flutter app featuring a basic button that generates a random dice roll upon being pressed.

## Version

- *Dart 3.2.6*
- *Flutter 3.16.9*

## Screenshot

## Building

These commands are intended for maintainers only.

### Android

Traditional APK

    flutter build apk

AppBundle for Google Play

    flutter build appbundle

### iOS

    flutter build ipa
